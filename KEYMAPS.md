# KEYMAPS

## Leader key
| Key   | Action           |
|-------|------------------|
| `,`   | Leader key (new) |

## Insert Mode
| Key   | Action                    |
|-------|---------------------------|
| `jj`  | Exit to Normal mode (Esc) |

## Normal Mode
| Key       | Action                                        |
|-----------|-----------------------------------------------|
| `<SPACE>` | Enter command mode (`:`)                      |
| `o`       | Insert new line below and stay in Normal mode |
| `O`       | Insert new line above and stay in Normal mode |
| `,w`      | Save file (`:w`)                              |

## Window Navigation
| Key     | Action                |
|---------|-----------------------|
| `<C-h>` | Move to left window   |
| `<C-j>` | Move to bottom window |
| `<C-k>` | Move to top window    |
| `<C-l>` | Move to right window  |

## Window Resizing
| Key         | Action                 |
|-------------|------------------------|
| `<C-UP>`    | Increase window height |
| `<C-DOWN>`  | Decrease window height |
| `<C-LEFT>`  | Increase window width  |
| `<C-RIGHT>` | Decrease window width  |

## Plugin Shortcuts
| Key         | Action                                             |
|-------------|----------------------------------------------------|
| `<LEADER>n` | Toggle NERDTree (`:NERDTreeToggle`)                |
| `<LEADER>f` | Reveal current file in NERDTree (`:NERDTreeFocus`) |
