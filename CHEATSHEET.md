# CHEATSHEET

## Global
| Key               | Action                  |
|-------------------|-------------------------|
| `:h[elp] keyword` | Open help for keyword   |
| `:sav[eas] file`  | Save file as            |
| `:clo[se]`        | Close current pane      |
| `:ter[minal]`     | Open a terminal window  |
| `K`               | Open a man page for word under the cursor |
